INPUT SOURCE

【输入源】原始直觉与原生痛感

输入的是你对现实世界抓耳挠腮、说不清道不明的“原始痛感”。

MODULE 01

【模块一】扮演“翻译官”兼“逻辑硬化器”

核心任务：把你脑子里模糊、零散、甚至带有个人情绪的“原始直觉”，转变成一段干净、高浓度、具备强解释力的“思想代码”。

经过模块一剥离情绪、跨界编译的疯狂压榨后，吐出来的必须是一个亮晶晶、硬邦邦、具备独特解释视角的深度逻辑假说（思想代码）。只有拿到了这个坚实的“子弹”，后面的模块二才能有物料去进行极限轰炸。

MODULE 02

【模块二】基于斯坦福小镇的多 Agent 模拟博弈

核心任务：基于斯坦福小镇完成模拟。“带着你的思想种子（深层结构 C）在特定地形里博弈”，系统强迫 Agent 带着深层结构 C 去做长期和短期的博弈行为规划。

基于其“记忆流-反思-规划（Memory-Reflection-Planning）”的主干设计，Agent 展开深度自驱博弈。需要严密定义路径和边界，避免无效的与核心逻辑无关的废话 token 费用。其反思产物即为 Crash Logs（边缘失效日志），直接投喂给模块三。

MODULE 03

【模块三】马尔科夫链与 Deep Think 动力链连缀

核心任务：利用 Deep Think 推理链，把模块二炸出来的碎屑和漏洞重新缝合，用一根刚性的“因果铁丝”从头穿到尾，打造自运转动力闭环。

通过构建一个马尔科夫链，确保输入变量 X 触发，必然 100% 带动变量 Y 发生，继而精准推动变量 Z。在逻辑内部消除一切“大概”、“可能”的模糊地带。一旦因果链成型，系统会精准计算出认知主体需要在哪些核心概念节点上进行“重仓布局”（分配主要的认知算力或践行精力）。

MODULE 04

【模块四】过拟合清算与波普尔生存死线网关

核心任务：像冰冷的断头台一样，强制为这条因果链划定“爆仓自毁死线”，剔除一切侥幸心理，通过的正式颁发“交付证书”，写进全局地图。

人的大脑和当前的 LLM 有个致命缺陷：过拟合、极度护短。我们会本能地编造各种“借口”去为错误辩护，让知识地图充满和稀泥的伪概念。模块四用一柄冰冷的哲学手术刀斩断所有自我防卫，通过者正式颁发证书并沉淀为抗打击资产，不通过者直接经由 LangGraph 路由全面打回输入层重构。
